# Test Mobile App

Aplicativo mobile desenvolvido no Android Studio apenas para testes e aprendizado.

## Sobre

Projeto criado para praticar desenvolvimento mobile, estrutura de aplicações Android e geração de arquivos `.apk`.

## Tecnologias

* Android Studio
* Java / Kotlin
* Android SDK

## Funcionalidades

* Interface mobile
* Testes de funcionamento em Android
* Geração de APK

## APK

O aplicativo foi desenvolvido somente para testes locais e validação de funcionalidades.

## Status

Projeto de estudos e testes.
